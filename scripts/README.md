# `curl` flags

- `-f` / `--fail` - Fail silently on HTTP errors (don't output error page)
- `-s` / `--silent` - Silent mode (don't show progress bar or errors)
- `-S` / `--show-error` - Show errors even in silent mode (used with -s)
- `-L` / `--location` - Follow redirects
